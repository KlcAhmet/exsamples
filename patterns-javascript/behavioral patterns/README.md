# Behavioral Patterns

Bu kalıplar, bir sistemdeki farklı nesneler arasında etkili iletişimi sağlar, nesnelere sorumluluklar atar ve hepsinin senkronize bilgiye sahip olmasını garanti eder.

Bu davranışsal kalıplar şunları içerir:

- Chain of Responsibility
- Interpreter
- Command
- Iterator
- Mediator
- Memento
- State
- Template Method*
- Strategy
- Visitor
- Observer
- Revealing Module
